# AlbumTTMLs
**TTML files for some selected albums.**

Raccolta di testi sincronizzati in formato **TTML** per album, mixtape e brani italiani da utilizzare su Spotify desktop tramite **Spicetify** e **Spicy Lyrics**.

L’obiettivo del progetto è rendere disponibili testi sincronizzati anche per brani locali, pubblicazioni non presenti su Spotify o canzoni senza lyrics complete.

> Questa repository non contiene e non distribuisce file audio.

---

## Contenuti

I pacchetti di TTML possono includere:

* testi sincronizzati riga per riga
* testi sincronizzati parola per parola
* testi di album e mixtape completi
* informazioni sulla versione audio usata per la sincronizzazione
* eventuali correzioni e aggiornamenti

Il tipo di sincronizzazione disponibile viene indicato nella cartella di ogni album.

---

## Requisiti

Per utilizzare i TTML servono:

* Spotify desktop (è necessario aver installato Spotify direttamente dal [sito ufficiale](https://www.spotify.com/it/download/windows/) e **NON** attraverso il Microsoft store)
* Spicetify
* Spicy Lyrics
* la patch di Spicy Lyrics per il caricamento dei TTML locali
* i file audio già presenti sul proprio computer

---

## Installazione

### 1. Installa Spicetify

Segui la documentazione ufficiale:

[Spicetify](https://spicetify.app/)

### 2. Installa Spicy Lyrics

Segui le istruzioni del progetto:

[Spicy Lyrics](https://spicylyrics.org/)

### 3. Applica la patch

La patch permette di utilizzare i file TTML locali con Spicy Lyrics.

[Guida all’installazione della patch](https://github.com/gingerik87/AlbumTTMLs/blob/main/patch-installation-guide.md)

### 4. Aggiungi i file locali a Spotify

Attiva i file locali dalle impostazioni di Spotify e aggiungi la cartella contenente i tuoi file audio.

Assicurati che titolo, artista, album e durata corrispondano alla versione usata per creare il TTML.

### 5. Scarica e associa i TTML

Apri la cartella dell’album desiderato, scarica i file e segui la guida dedicata.

[Guida all’utilizzo dei TTML](INSERIRE_LINK)

## Problemi e correzioni

Se trovi:

* righe fuori tempo
* errori nel testo
* parole mancanti
* problemi con una versione audio specifica
* incompatibilità con nuovi aggiornamenti

puoi aprire una Issue indicando album, traccia, timestamp (momento preciso della canzone) e durata del file audio utilizzato.
